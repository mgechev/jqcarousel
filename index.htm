About


jQCarousel is a jQuery UI plugin distributed under the General Public License, developed by Minko Gechev, used for creating carousel galleries. It's not using canvases or SVG so it has big browser compatibility. It has rich API for such a small plugin. You can find it at GitHub. The plugin is under construction so it's performance and features are extending constantly. If you wish to help improving the gallery you can fork the project in GitHub or send me advices via any of the contact methods provided in the section below.
